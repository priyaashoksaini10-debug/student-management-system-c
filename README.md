# 🎓 Student Management System in C

## 📌 Overview
This project is a console-based Student Management System developed in C language.  
It demonstrates file handling, structures, and modular programming concepts.

## 🚀 Features
- Admin login system
- Add student records
- View all students
- Search student by roll number
- Update student details
- Delete student records
- Data persistence using file handling

## 🛠️ Technologies Used
- C Language
- File Handling
- Structures
- Functions
- Sorting Algorithms (optional upgrade)

## 📚 Learning Outcomes
- Strong understanding of C programming concepts
- Real-world CRUD operations
- File handling in C
- Logical problem solving
